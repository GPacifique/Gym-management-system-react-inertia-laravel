### README Intro Version

# GymMate - Gym Management System

GymMate is a scalable gym management platform developed using Laravel 12, React.js, Inertia.js, and MySQL. It helps fitness centers manage members, staff, trainers, subscriptions, attendance, payments, workout programs, and daily operations through a clean and modern dashboard.

## Features

* Multi-role authentication and authorization
* Super Admin gym management
* Member registration and profiles
* Trainer and staff management
* Attendance tracking
* Subscription and payment management
* Workout and nutrition programs
* Notifications (Email & SMS ready)
* Analytics dashboards
* Responsive React frontend
* RESTful backend architecture

## Tech Stack

* Laravel 12
* React.js
* Inertia.js
* MySQL
* Tailwind CSS
* Spatie Permission Package

## Roles

* Super Admin
* Gym Owner
* Manager
* Trainer
* Receptionist
* Member
